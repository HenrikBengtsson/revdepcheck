# Cancel a running cloud run

Cancel a running cloud run

## Usage

``` r
cloud_cancel(job_name = cloud_job())
```

## Arguments

- job_name:

  The job name, as returned by
  [`cloud_check()`](https://revdepcheck.r-lib.org/reference/cloud_check.md).
